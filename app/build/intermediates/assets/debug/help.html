1. Introduction

VQEG Image Quality Evaluation Tool (VIQET) is an objective, no-reference photo quality evaluation tool. VIQET is free and open source tool designed to evaluate quality of consumer photos. In order to perform photo quality evaluation, VIQET requires a set of photos from the test device. It estimates an overall Mean Opinion Score (MOS) for a device based on the individual image MOS scores in the set. The estimated MOS for each photo is based on a number of image quality features and statistics extracted from the test photo. The mapping from extracted features to MOS is based on psychophysics studies that were conducted to create a large dataset of photos and associated subjective MOS ratings. The studies were used to learn a mapping from quantitative image features to MOS. The estimated MOS by VIQET falls in a range of 1 to 5, where 1 corresponds to a low quality rating and 5 corresponds to excellent quality. More information about VIQET and other VQEG projects is available at www.VQEG.org

2. VIQET Software

VIQET users have two tool options: VIQET Desktop application and VIQET App (Android*). Both tools utilize the same image analysis and MOS estimation models and produce exactly same results for a given photo/photo set. The VIQET desktop application is designed to run on a Windows* based PC and allows users to evaluate quality of photos from all mobile devices. All images are evaluated locally in the VIQET desktop application and the analysis system is not required to be connected to the Internet. The VIQET App is designed to run on the Android* mobile device under test and the image analysis is performed by a VIQET cloud service. VIQET App requires the test device to be connected to the Internet to perform the photo quality evaluation.

VIQET Desktop Application: The latest version of the VIQET Desktop Application software is available as a free download from https://github.com/VIQET/VIQET-Desktop/releases

VIQET App (Android): The latest version of the VIQET App is available as a free download from Google* Play Store. VIQET App for iOS* and Windows* mobile devices are currently not available.

3. Using VIQET App to analyze photos

There are 2 steps in this process. First is capturing test photos according VIQET methodology, and second, using the tool to analyze

them.
 

General Guidelines for Capturing Test Photos

VIQET requires test photos in four specific input categories

1. Outdoor Day - Landscape

2. Indoor – Wall Hanging

3. Indoor – Arrangements

4. Outdoor Night - Landmark

     The following general guidelines for capturing test photos are crucial to obtain reliable VIQET results.  Please ensure that you follow the guidelines below as closely as you can each time you take a photo for any of the four categories explained in the next section. 

i. Ensure that you capture 5 photos in each of the test photo categories mentioned in the next section

ii. Avoid direct light sources in all scenes

iii. Avoid moving and animated objects such as cars and people

iv. Avoid reflective surfaces such as glass or mirrors

v. Avoid photographer shadow

Note: If you are using VIQET to compare different devices, please capture the same scene by the devices. That means that you may have to move closer or further from the scene in order to capture the same photo. Also, ensure that you do not show new objects in the photo as you are switching devices to capture the same scene. VIQET App guides you in capturing (or choosing previously captured) required test photos and analyzing them.

 

    

    

 

 Analyze Test Photos

1. Launch the VIQET App. You are now on the home screen. To create a new test, click on the ‘Start New Test’ button and enter

a test name. Use a name that will help you identify the test content.

2. You are now on the Add Photos screen. Add required number of photos in each category by following instructions on the

screen. You can choose an existing picture or capture a new photo from within the VIQET app.

3. The added photos are sent to the VIQET cloud service for analysis and results are displayed upon completion. You can add

all the required photos while the added images are being processed.

4. After analyzing all the required photos in the set, the test results are displayed. From this screen, you can view the Overall

results and detailed results for individual Categories and Images.

   

Note: You can choose to view the Sharpness map and an RGB histogram for each photo

5. Index

1. Resolution: refers to the total number of little squares (pixels) that make up an image. The higher the resolution, the higher the number of pixels.

2. Saturation: refers to how vivid and intense a color is. An image with poor color saturation will look washed out or faded. When a color's saturation level is reduced to 0, it becomes a shade of gray.

3. Color Warmth: refers to the tint of the overall image. Images with a bluish tint are considered to have cool colors. Images with an orange-reddish tint are considered to have warm colors.

4. Dynamic Range: is the range between the lightest and darkest regions in an image while maintaining details of an image in both the lightest and darkest spots (represented in shades of grey).

5. Illumination: refers to how well-lit an image is. An image is considered well-lit if it is bright and has a sufficient amount of detail.  Its values ranges from 0 - 255.

6. Multi-scale edge acutance: refers to how sharp the outline of objects in an image are and how many edges were detected in the scene. The sharper the image, the higher the multi-scale edge acutance feature.

7. Multi-scale texture acutance: refers to the level of activity and detail in the scene. The higher the level of detail in the scene the higher this feature value is.

8. Noise signature index: refers to how noisy or grainy an image is. This feature value ranges from 0 to 589. The higher the index, the grainier an image is.

9. % Over-exposed: refers to the percentage of the image area that is covered in white. This feature ranges from 0 to 100. A higher percentage value indicates a larger area of an image is over-exposed.

10. % Under-exposed: refers to the percentage of the image area that is covered in black. This feature ranges from 0 to 100. A higher percentage value indicates a larger area of an image is under-exposed.

11. Lux: refers to a measure of light intensity that hits or passes through a surface, as perceived by the human eye.
